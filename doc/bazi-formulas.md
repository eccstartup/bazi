# 八字排盘计算公式

## 1. 年柱

```
年天干 = TIAN_GAN[(year - 4) % 10]
年地支 = DI_ZHI[(year - 4) % 12]
```

**原理**：公元 4 年为甲子年，以此为基准，差值取余推算。

**⚠️ 局限**：当前实现按公历年直接计算，未按节气（立春）换年。传统命理中立春后才算新的一年。

验算：`2024-4=2020, 2020%10=0→甲, 2020%12=0→辰 → 甲辰` ✅

---

## 2. 月柱

```
五虎遁表（年干 → 寅月天干）:
  甲己 → 丙寅, 乙庚 → 戊寅, 丙辛 → 庚寅, 丁壬 → 壬寅, 戊癸 → 甲寅
  yinCorr = [2, 4, 6, 8, 0]  （对应甲→丙[索引2], 乙→戊[4], 丙→庚[6], 丁→壬[8], 戊→甲[0]）

start = yinCorr[((year - 4) % 10) % 5]
月天干 = TIAN_GAN[(start + month - 1) % 10]
月地支 = DI_ZHI[(month + 1) % 12]
```

**原理**：年干定寅月天干（五虎遁法），顺推至目标月份。

**⚠️ 局限**：当前按月数直接计算，未按节气（立春二月寅、惊蛰三月卯…）精确排盘。

验算：1986 丙寅年 5 月 → `start=6, (6+5-1)%10=0→癸, (5+1)%12=6→巳 → 癸巳`  
（当前实现输出 甲午，月支公式 `(month+1)%12` 下 5 月→午，start 因年干丙=2 而非 6，细节见源码）

## 3. 日柱

```
基准日: 2000-01-01 = 甲子
diff = round((目标日期 - 基准日) / 86400000)

日天干 = TIAN_GAN[((diff % 60) + 60) % 60 % 10]
日地支 = DI_ZHI[((diff % 60) + 60) % 60 % 12]
```

**原理**：以 60 日为一个甲子周期，相对 2000-01-01 甲子的天数偏移量直接映射。

验算：`2000-01-01 → diff=0 → 甲子 ✅`  
`2021-01-01 → diff=7671 → 7671%10=1→乙, 7671%12=3→卯 → 乙卯 ✅`

### 4.2 五鼠遁（日干 → 子时天干）

```
甲己 → 甲子, 乙庚 → 丙子, 丙辛 → 戊子, 丁壬 → 庚子, 戊癸 → 壬子
公式: start = (日干索引 % 5) * 2
时天干 = TIAN_GAN[(start + 时辰索引) % 10]
时地支 = 时辰名
```

**原理**：日干定子时天干，顺推至目标时辰。

验算：甲日子时 → `start=0, (0+0)%10=0→甲, 子 → 甲子 ✅`  
甲日午时 → `(0+6)%10=6→庚, 午 → 庚午 ✅`  
乙日丑时 → `(1%5)*2=2, (2+1)%10=3→丁, 丑 → 丁丑 ✅`

## 8. 神煞（驿马/桃花）

以**日支**为基准推算各柱地支：

```
驿马:  寅午戌→申, 申子辰→寅, 巳酉丑→亥, 亥卯未→巳
桃花:  寅午戌→卯, 申子辰→酉, 巳酉丑→午, 亥卯未→子
```

当前仅实现了驿马和桃花两种神煞。

## 已知局限

1. **年柱**：不按立春换年，用公历年边界直接推算
2. **月柱**：不按节气分月，直接用月份数推算（1 月→寅月）
3. **日柱**：以 2000-01-01 为基准相对推算，不同历算系统可能有 ±1 日偏差
4. **时柱**：23 点后未切换次日日柱
5. **神煞**：仅实现驿马、桃花，未包含天乙贵人、文昌、羊刃等常见神煞
