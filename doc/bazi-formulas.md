# 八字排盘计算公式与原理

当前排盘引擎已通过 `lunar-javascript` 库进行了重构，全面支持中国传统的、精确的八字（四柱）命理计算。以下为正确的排盘公式与天文历法原理。

---

## 1. 年柱（以“立春”为界）

在传统八字中，新一年的开始**既不是公历的 1 月 1 日，也不是农历的大年初一，而是“立春”节气交节的时刻**。

*   **计算规则**：
    *   在立春时刻之前出生的人，年干支属于前一年。
    *   在立春时刻及之后出生的人，年干支属于新一年。
*   **公式推算**：
    *   以公元前 2697 年（黄帝纪元元年，甲子年）为起点进行干支纪年循环。
    *   基准年柱计算：`年天干 = TIAN_GAN[(year - 4) % 10]`，`年地支 = DI_ZHI[(year - 4) % 12]`（仅在立春后适用，立春前需将 `year` 减 1 演算）。
*   **验算**：
    *   `2000-01-01`：由于在 2000 年立春（2月4日左右）之前，属于 1999 己卯年 → 年柱为 **己卯** ✅。
    *   `1986-05-26`：在立春之后 → 年柱为 **丙寅** ✅。

---

## 2. 月柱（以“节气”分界）

月柱的地支（月支）完全由**十二节气**决定，每两个节气之间为一个完整的命理月份。

*   **月支划分规则**：
    *   **寅月**：立春（2月4日左右）到惊蛰
    *   **卯月**：惊蛰（3月5日左右）到清明
    *   **辰月**：清明（4月4日左右）到立夏
    *   **巳月**：立夏（5月5日左右）到芒种
    *   **午月**：芒种（6月5日左右）到小暑
    *   **未月**：小暑（7月7日左右）到立秋
    *   **申月**：立秋（8月7日左右）到白露
    *   **酉月**：白露（9月7日左右）到寒露
    *   **戌月**：寒露（10月8日左右）到立冬
    *   **亥月**：立冬（11月7日左右）到大雪
    *   **子月**：大雪（12月7日左右）到小寒
    *   **丑月**：小寒（1月5日左右）到立冬
*   **月干确定（五虎遁法）**：
    根据年天干，推算出寅月（正月）的月干，然后顺推。
    ```
    甲己年 → 丙寅起    乙庚年 → 戊寅起    丙辛年 → 庚寅起
    丁壬年 → 壬寅起    戊癸年 → 甲寅起
    ```
*   **验算**：
    *   `1986-05-26`（丙寅年）：处于立夏与芒种之间，属**巳月**。因年干为丙，按“丙辛年庚寅起”，正月庚寅、二月辛卯、三月壬辰、四月**癸巳** → 月柱为 **癸巳** ✅。

---

## 3. 日柱（六十甲子顺推）

日柱由六十甲子依次循环纪日，每日一换（以子时即晚上 23:00 为一天的分界线）。

*   **基准日（Epoch）**：
    *   公历 **2000-01-01** 实际是 **戊午日**。
    *   戊午在六十甲子中的索引为 **54**（甲子为 0）。
*   **公式推算**：
    ```
    diff = round((目标日期 - 2000-01-01) / 86400000)
    offset = ((diff + 54) % 60 + 60) % 60
    日天干 = TIAN_GAN[offset % 10]
    日地支 = DI_ZHI[offset % 12]
    ```
*   **验算**：
    *   `2000-01-01`：`diff = 0` → 索引为 `54` → **戊午日** ✅。
    *   `1986-05-26`：`diff = -4968` → `offset = 6` → **庚午日** ✅。
    *   `2024-06-10`：`diff = 8927` → `offset = 41` → **乙巳日** ✅。

---

## 4. 时柱（以“五鼠遁”推算）

一日分为十二个时辰，每时辰占两小时。

### 4.1 时辰地支划分

| 时辰 | 子 | 丑 | 寅 | 卯 | 辰 | 巳 | 午 | 未 | 申 | 酉 | 戌 | 亥 |
|------|----|----|----|----|----|----|----|----|----|----|----|----|
| 范围 | 23-1 | 1-3 | 3-5 | 5-7 | 7-9 | 9-11 | 11-13 | 13-15 | 15-17 | 17-19 | 19-21 | 21-23 |

### 4.2 时干确定（五鼠遁法与早晚子时）

根据**出生日的天干**推算子时的时干，然后顺推。
```
甲己日 → 甲子起    乙庚日 → 丙子起    丙辛日 → 戊子起
丁壬日 → 庚子起    戊癸日 → 壬子起
```

*   **晚子时（夜子时）规则**：
    若出生在 23:00 - 24:00 之间，日天干依然采用当前日的天干，但时柱的干支需要**按照下一天的天干来推算**（因为时支已跨入子时，属于新一天的首个时辰）。
*   **验算**：
    *   `2000-01-01 23:30`（戊午日，夜子时）：
        *   日柱仍为 **戊午**。
        *   时柱按下一天（己未日，己日）的子时推算：己日对应“甲子起”，因此时柱为 **甲子** ✅。

## 重构后的优势（已消除的局限）

*   **年柱**：支持精确到“分秒级”的立春节气分界线，解决了公历/农历新年交替期的换年错误。
*   **月柱**：完美按二十四节气精确分月，月柱的干支不再受公历月数影响。
*   **日柱**：基准点修正为真实的“戊午”纪日，解决了天数偏移的错误。
*   **时柱**：完美支持“晚子时（夜子时）”排盘逻辑。
